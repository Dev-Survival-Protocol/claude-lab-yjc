2026-03-24, memory.md (claude-code-basic/) 팩트체크 수행. 원본: https://code.claude.com/docs/ko/memory

**Why:** 요약 문서와 원본 공식 docs 간 사실 관계 검증 요청

**How to apply:** 향후 이 파일 재검토 시 아래 오류 이력 참고

## 발견된 오류 패턴

1. **사용 목적 표현 누락**: 비교표 자동 메모리 "사용 목적" 셀에서 "Claude가 발견한"이라는 수식어 누락. 원본: "Claude가 발견한 선호도"
2. **우선순위 방향 혼란**: 사용자 수준 규칙 섹션에서 "프로젝트 규칙보다 낮은 우선순위로 로드된다"는 결론은 맞지만, 원본은 "프로젝트 규칙 **전에** 로드되어 **프로젝트 규칙에 더 높은 우선순위를 제공**"이라는 메커니즘으로 설명. 로드 순서와 우선순위 개념이 혼재되어 있음.
3. **autoMemoryDirectory 허용 범위 누락**: "프로젝트 설정에서는 불가"만 언급, "사용자/로컬/정책 설정에서 허용" 누락
4. **/memory UI 토글 오해 소지**: `/memory` 명령이 토글 명령처럼 기술되어 있으나, 실제로는 UI를 열고 내부에서 토글 사용
5. **MEMORY.md 200줄 제한 범위 미구분**: 200줄 제한이 MEMORY.md에만 적용되고 CLAUDE.md는 전체 로드된다는 중요한 구분이 요약 문서에 없음
